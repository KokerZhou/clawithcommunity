# Clawith Community Images

Community-published Docker images for [Clawith](https://github.com/dataelement/Clawith).

## Images

Docker Hub:

```text
clawithcommunity/clawith-backend
clawithcommunity/clawith-frontend
```

GitHub Container Registry:

```text
ghcr.io/kokerzhou/clawith-backend
ghcr.io/kokerzhou/clawith-frontend
```

## Tags

Use `latest` for the upstream release marked as latest by GitHub.

Release-specific tags are also published for upstream releases, using the
upstream release tag as the version label.

## Source

These images are built from published upstream Clawith releases, including
prereleases. Source code and project documentation live in the upstream
repository:

```text
https://github.com/dataelement/Clawith
```
